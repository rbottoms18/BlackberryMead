☼ BlackberryMead ☼
------------------------------------------------------------------------------------------------------

Light-weight game engine built on top of the MonoGame Framework.
Created by Romato.
** requires C# Version 8.0 or greater **

Disclaimer: This engine library is developed for and alongside an independent game and is not intended 
(yet) as a a stand alone library. Features may be incomplete or buggy. Features will be added
whenever they are needed in the main game, not for completeness of this library. No gaurantee is made that 
future versions of this library will be compatible with previous versions.
